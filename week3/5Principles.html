Affordance



The website is simple and setup in a manner where there is no false buttons. Every attribute on the page has a function and the buttons are clearly labeled.

Mapping



Wikipedia is setup with the control on the top and left side of the page, this makes it easy to locate buttons/links. It also simplifies the navigation process.

Chunking



The websie breaks the different types of information into multiple sections, this layout allows the eye to scan between the sections easily.

Hick's Law



This simply design website decreases the number of distractions for the user. The user is not confronted with a lot of decisions when trying to navigate thw website.

Modularity



This site breaks up the content into small easy to navigate sections. This would make the website easy to edit if the developer decided to add a new section or more content.
